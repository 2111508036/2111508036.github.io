      概述

      您可以轻松更改用户界面的文本，以便将其翻译成特定语言。我们为开发人员提供了预定义的语言，只需加载语言集并更改一个设置即可应用，开发人员还可以使用现有语言文件的模板创建自己的语言集。

      TIP

      要正确显示 RTL 语言，请配置 Handsontable 的布局方向.

      加载准备好的语言文件

      要正确使用国际化功能，需要加载语言集。在 Handsontable 文件之后加载语言集非常重要。您可以根据 UMD 标准创建必要的文件：

      1. ES modules (ESM)
      import Handsontable from 'handsontable/base';
import { registerLanguageDictionary, deDE } from 'handsontable/i18n';

registerLanguageDictionary(deDE);

const hot = new Handsontable(container, {
  language: deDE.languageCode,
});
      1. CommonJS (CJS)
      const Handsontable = require('handsontable/base').default;
const { registerLanguageDictionary, deDE } = require('handsontable/i18n');

registerLanguageDictionary(deDE);

const hot = new Handsontable(container, {
  language: deDE.languageCode,
});
      1. Universal Module Definition (UMD)

      以这种方式包含的语言在加载文件后可立即使用。每个文件都包含一个 UMD 加载器，可在全局/外部上下文中查找 Handsontable。如果 Handsontable 可用，它就会在适当的上下文中注册自己。

      <script type="text/javascript" src="dist/handsontable.full.js"></script>
<script type="text/javascript" src="dist/languages/de-DE.js"></script>
<script>
  const hot = new Handsontable(container, {
    language: 'de-DE',
  });
</script>

      演示

      要查看已翻译的上下文菜单，请右键单击单元格。

      加载 Handsontable 后加载了语言文件。

        import Handsontable from 'handsontable';
import { registerLanguageDictionary, deDE } from 'handsontable/i18n';
import 'handsontable/dist/handsontable.full.min.css';

registerLanguageDictionary(deDE);

const container = document.querySelector('#example1');
const data = [
  ['Lorem', 'ipsum', 'dolor', 'sit', '12/1/2015', 23],
  ['adipiscing', 'elit', 'Ut', 'imperdiet', '5/12/2015', 6],
  ['Pellentesque', 'vulputate', 'leo', 'semper', '10/23/2015', 26],
  ['diam', 'et', 'malesuada', 'libero', '12/1/2014', 98],
  ['orci', 'et', 'dignissim', 'hendrerit', '12/1/2016', 8.5]
];

const hot = new Handsontable(container, {
  data,
  contextMenu: true,
  height: 'auto',
  language: 'de-DE',
  licenseKey: 'non-commercial-and-evaluation'
});

        可用语言列表

        默认情况下，Handsontable 使用英语-美国语言国家集（en-US 代码）创建 UI 元素文本。不过，它也可以像其他 "非标准 "语言文件一样使用，因此 en-US.js 文件可以在/dist/languages, /languages 和 /src/languages文件夹中找到。目前，我们还分发了额外的语言国家文件：

        • ar-AR.js 表示阿拉伯语 - 全球（ar-AR 代码）。要按原意呈现这种语言，请将布局方向设置为 RTL。
        • cs-CZ.js 表示捷克 - 捷克 (cs-CZ code).
        • de-CH.js 表示德国 - 瑞士 (de-CH code).
        • de-DE.js 表示德国 - 德国 (de-DE code).
        • es-MX.js 表示西班牙语 - 墨西哥 (es-MX code).
        • fr-FR.js 表示法语 - 法国 (fr-FR code).
        • it-IT.js 表示意大利语 - 意大利 (it-IT code).
        • ja-JP.js 表示日语 - 日本 (ja-JP code).
        • ko-KR.js 表示韩语 - 韩国 (ko-KR code).
        • lv-LV.js 表示拉脱维亚语 - 拉脱维亚 (lv-LV code).
        • nb-NO.js 表示挪威语(Bokmål) - 挪威 (nb-NO code).
        • nl-NL.js 表示荷兰语 - 荷兰 (nl-NL code).
        • pl-PL.js 表示波兰语 - 波兰 (pl-PL code).
        • pt-BR.js 表示葡萄牙语 - 巴西 (pt-BR code).
        • ru-RU.js 表示俄语 - 俄罗斯 (ru-RU code).
        • sr-SP.js 表示塞尔维亚语（拉丁语） - 塞尔维亚 (sr-SP code).
        • zh-CN.js 表示中文简体 - 中国 (zh-CN code).
        • zh-TW.js 表示中文繁体 - 中国台湾 (zh-TW code).

        创建自定义语言

        您可以为您的实施创建自定义语言集，也可以共享这些语言集，因为它们很容易适用于任何 Handsontable 实施。

        语言文件

        对我们来说，社区是图书馆发展的重要组成部分，这一点非常重要。我们鼓励您创建并分享您的翻译作品！

        其他语言文件应放置在 Handsontable 资源库的 src/i18n/languages 文件夹中，其名称应与所选语言代码相对应（如下所述，例如：es-VE.js）。您可以通过向我们发送拉取请求，将您的翻译加入 Handsontable 库。重要的是，您的改动不能在 /languages 和 /dist/languages 目录中进行！我们将生成最终文件并放置在那里。之后，你就可以在 Handsontable 中使用这些语言了。

        您可以在本段底部看到示例语言的完整模板。我们以默认语言包为基础 (opens new window)。文件创建过程的部分内容如下。

        1. 文件开头应有注释，包含翻译作者（用逗号分隔，例如：作者：克里斯-维克、约翰-凯尔）、"最后更新"日期（格式为：mmm dd, yyyy，例如：最后更新：2017 年 1 月 1 日）和说明。

          /**
* @preserve
* Authors: Chris Wick, John Kyle
* Last updated: Nov 15, 2017
*
* Description: Definition file for Spanish - Venezuela language-country.
*/

        2. 现在，导入要在翻译中使用的字典键。

          import * as C from '../constants';

        3. 语言字典对象应包含语言代码键（格式为：两个小写字母、连字符、两个大写字母，例如：languageCode：例如：languageCode: 'es-PY'），它将决定在 Handsontable 设置和字典键的语言属性中使用的语言代码及其相应的翻译。

          const dictionary = {
  languageCode: 'es-VE',
  [C.CONTEXTMENU_ITEMS_ROW_ABOVE]: 'Insertar fila arriba',
  ...
}

        4. 最后，对创建的字典进行默认导出。

          export default dictionary;

        5. 一个简单的示例语言字典可以如下所示。构建过程将生成 /languages 和 /dist/languages 文件夹。这些本地化文件可以包含在本节中。加载这些文件后，就可以使用这些语言了。将 Handsontable 的语言设置更改为 es-VE 即可。

          /**
* @preserve
* Authors: Chris Wick, John Kyle
* Last updated: Nov 15, 2017
*
* Description: Definition file for Spanish - Venezuela language-country.
*/
import * as C from '../constants';

const dictionary = {
  languageCode: 'es-VE',
  [C.CONTEXTMENU_ITEMS_ROW_ABOVE]: 'Insertar fila arriba',
};

export default dictionary;

        6. 在 src/i18n/languages/index.js 文件中导入已创建的文件，然后像下面的代码段一样导出（按字母顺序键入）。




           






           



          import deCH from './de-CH';
import deDE from './de-DE';
import enUS from './en-US';
import esVE from './es-VE';
import plPL from './pl-PL';

export {
  deCH,
  deDE,
  enUS,
  esVE,
  plPL
};

        7. 瞧！您创建了一种语言，既可以自己使用，也可以与他人共享。我们会等待至少 5 个来自用户的积极反馈，以接受创建的拉取请求。

        本地语言

        您可以注册不属于 Handsontable 软件包的语言字典。为此，请使用静态 Handsontable.languages.registerLanguageDictionary 方法和静态常量 Handsontable.languages.dictionaryKeys，下一节将对此进行简要介绍。

        const C = Handsontable.languages.dictionaryKeys;

Handsontable.languages.registerLanguageDictionary({
  languageCode: 'morse',
  // 您的莫尔斯电码译文
  [C.FILTERS_BUTTONS_OK]: '--- -•-'
});

        在翻译中使用自定义键

        您可以注册包含自定义键的语言字典。这些条目可以像其他键一样使用，因此您不必局限于使用我们预定义的常量（这些常量存在于 src/i18n/constants.js 文件中，可以通过 Handsontable.languages.dictionaryKeys 别名访问）。

        const enUSDictionary = Handsontable.languages.getLanguageDictionary('en-US');

enUSDictionary.customKey = 'Hello world';

Handsontable.languages.registerLanguageDictionary(enUSDictionary); // re-registration
Handsontable.languages.getTranslatedPhrase('en-US', 'customKey'); // 'Hello world'

        静态 Handsontable 方法和属性

        Handsontable 有一些与语言相关的静态方法和属性。它们存储在全局 Handsontable 变量的语言键中。下面将对所有方法和属性进行说明。

        获取特定语言代码的语言词典

        Handsontable.languages.getLanguageDictionary(languageCode: String)

        返回: Object

        获取已注册的语言词典

        Handsontable.languages.getLanguagesDictionaries()

        返回: Array

        获取指定字典键的短语

        Handsontable.languages.getTranslatedPhrase(languageCode: String, dictionaryKey: String, extraArguments: Mixed)

        返回: Object

        为特定语言代码注册语言词典

        Handsontable.languages.registerLanguageDictionary(languageCodeOrDictionary: Mixed, dictionary: Object)

        返回: Object

        词典常量

        Handsontable.languages.dictionaryKeys

        包含: Object
