      什么是 Handsontable?

      Handsontable（发音为 "hands-on-table"）是一个 JavaScript 数据表格组件,能为您的应用程序带来众所周知的电子表格的外观和感觉.

      成千上万的商业应用程序都依赖 Handsontable 来输入、编辑、验证和清理数据,这些数据来自数据库和 API 等远程资源,或 HTML 文档、Excel 文件、Google Sheets 和手动输入.

      开始使用 Handsontable

      要直接进入示例代码,请在 CodeSandbox 上打开演示应用程序:

      • JavaScript demo (opens new window)
      • React demo (opens new window)
      • Angular demo (opens new window)
      • Vue 2 demo (opens new window)
      • TypeScript demo (opens new window)

      然后,继续连接数据并配置Handsontable 的内置功能.要实现更高级的功能,请使用 Handsontable的API.

      支持的框架

      Handsontable通过封装器支持流行的JavaScript框架.它还提供了一个TypeScript声明文件,对API方法, 钩子, 和选项进行了标准化.

      查看Handsontable的React, Angular, Vue 2, 和Vue 3文档.

      Handsontable可以用来做什么?

      将 Handsontable 视为一个可扩展的框架,可让您快速构建表格式、面向数据的用户界面.Handsontable 可帮助开发人员解决实际问题.举几个例子

      • 在内部财务应用中,基于 Handsontable 的可编辑表格简化了从 Excel 和 Google Sheets 中导入手工挑选的数据的过程.
      • 在一家建筑公司的软件中,使用 Handsontable 建立的交互式数据表可帮助用户修改规范和标准表.
      • 在医院,Handsontable 可帮助跟踪和管理物资.
      • 一家移动游戏公司使用 Handsontable 来简化开发的某些环节.
      • 在项目管理软件中,Handsontable 允许管理者每周收集团队和客户的反馈意见.

      软件类型

      例如,Handsontable 的内置功能和可定制性使其适用于不同行业和不同类型的软件:

      • 数据建模应用程序
      • 资源规划软件
      • ERP软件
      • 建筑数字化平台
      • 佣金自动化工具
      • 知识管理系统
      • 公民举报平台
      • 数据管理系统
