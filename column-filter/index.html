      概述

      过滤功能可让您根据特定条件快速找到所需的信息。这让数据分析变得更简单、更快捷，尤其是在处理大型数据集时。

      Handsontable 的内置过滤界面与 Excel 相似，因此即使是非技术用户也能直观地使用。如果你想实现自己的界面，还可以使用 Handsontable 的应用程序接口（API）以编程方式轻松过滤数据。

      您可以按值筛选数据，也可以使用内置条件，每种可用的列类型都有不同的内置条件。

      过滤演示

      点击其中一个列菜单按钮 (▼)，通过选择值或基于条件的标准进行筛选。

      过滤后，列宽会重新调整为屏幕上显示的最长值。要禁用这种行为，请设置固定列宽.

        启用过滤

        要启用所有列的筛选界面，需要做两件事：

        1. 将filters选项设为 true。
        2. 将 columnMenu 选项设置为 true，从而启用界面。

        如果您计划使用 API 创建自己的自定义界面来进行筛选，那么启用不带界面的筛选器选项会很有用。

        const configurationOptions = {
  // 启用过滤功能
  filters: true,
  // 启用列菜单
  dropdownMenu: true,
};

        默认情况下，列菜单会显示过滤界面以及其他默认项目，如插入列左侧。要只显示筛选界面，可在配置中传递筛选项数组。

              初始化时过滤数据

// 添加过滤器输入监听器
filterField.addEventListener('keyup', function (event) {
  const filters = handsontableInstance.getPlugin('filters');
  const columnSelector = document.getElementById('columns');
  const columnValue = columnSelector.value;

  filters.removeConditions(columnValue);
  filters.addCondition(columnValue, 'contains', [event.target.value]);
  filters.filter();

  handsontableInstance.render();
});

                          以编程方式控制过滤

                          您可以使用 Handsontable 的hooks和 API methods在表格运行时控制过滤。这样，你就可以根据特定条件启用或禁用过滤功能。例如，你可以在表格外创建一个用户界面来管理 Handsontable 的过滤行为。

                          以编程方式启用或禁用过滤功能

                          要以编程方式启用或禁用过滤功能，请使用 updateSettings() 方法。

                          handsontableInstance.updateSettings({
  // 启用过滤功能
  filters: true,
  // 启用列菜单
  dropdownMenu: true,
});

handsontableInstance.updateSettings({
  // 关闭过滤功能
  filters: false,
});

                          您还可以启用或禁用特定列的筛选功能。例如，只对第一列启用过滤功能：

                          handsontableInstance.updateSettings({
  // 启用过滤功能，适用于所有列
  filters: true,
  // 启用所有列的列菜单
  // 但只显示 "按值筛选 "列表以及 "确定 "和 "取消 "按钮
  dropdownMenu: {
    items: {
      filter_by_value: {
        // 隐藏除第一列外所有列的 "按值筛选 "列表
        hidden() {
          return this.getSelectedRangeLast().to.col > 0;
        },
      },
      filter_action_bar: {
        // 隐藏除第一列以外所有列的 "确定 "和 "取消 "按钮
        hidden() {
          return this.getSelectedRangeLast().to.col > 0;
        },
      },
    },
  },
});
