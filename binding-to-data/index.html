      兼容的数据类型

      嵌套数组

      对于更多类似表格的场景，嵌套数组是一个不错的选择，在这些场景中，您需要向最终用户提供操作表格的权限，例如插入列、删除行、装饰单元格等。.

        import Handsontable from 'handsontable';
import 'handsontable/dist/handsontable.full.min.css';

const container = document.querySelector('#example1');
const data = [
  ['', 'Tesla', 'Nissan', 'Toyota', 'Honda', 'Mazda', 'Ford'],
  ['2017', 10, 11, 12, 13, 15, 16],
  ['2018', 10, 11, 12, 13, 15, 16],
  ['2019', 10, 11, 12, 13, 15, 16],
  ['2020', 10, 11, 12, 13, 15, 16],
  ['2021', 10, 11, 12, 13, 15, 16]
];

const hot = new Handsontable(container, {
  data,
  startRows: 5,
  startCols: 5,
  height: 'auto',
  width: 'auto',
  colHeaders: true,
  minSpareRows: 1,
  licenseKey: 'non-commercial-and-evaluation'
});

        具有选择性显示列的嵌套数组

        以下示例显示如何使用嵌套数组来选择性显示列。此场景使用与上一个示例相同的数据源，这次省略了表格中的Tesla列.

          import Handsontable from 'handsontable';
import 'handsontable/dist/handsontable.full.min.css';

const container = document.querySelector('#example2');
const data = [
  ['', 'Tesla', 'Nissan', 'Toyota', 'Honda', 'Mazda', 'Ford'],
  ['2017', 10, 11, 12, 13, 15, 16],
  ['2018', 10, 11, 12, 13, 15, 16],
  ['2019', 10, 11, 12, 13, 15, 16],
  ['2020', 10, 11, 12, 13, 15, 16],
  ['2021', 10, 11, 12, 13, 15, 16]
];

const hot = new Handsontable(container, {
  data,
  colHeaders: true,
  minSpareRows: 1,
  height: 'auto',
  width: 'auto',
  columns: [
    { data: 0 },
    // skip the second column
    { data: 2 },
    { data: 3 },
    { data: 4 },
    { data: 5 },
    { data: 6 }
  ],
  licenseKey: 'non-commercial-and-evaluation'
});

const hot = new Handsontable(container, {

            以列作为函数的对象数组

            您可以将列 配置选项设置为函数。当你想更动态地绑定数据时，这是一种很好的做法.

              import Handsontable from 'handsontable';
import 'handsontable/dist/handsontable.full.min.css';

const container = document.querySelector('#example4');
const data = [
  { id: 1, name: {first: 'Ted', last: 'Right'}, address: '' },
  { id: 2, address: '' },// Handsontable will create missing properties on demand
  { id: 3, name: {first: 'Joan', last: 'Well'}, address: '' }
];

const hot = new Handsontable(container, {
  data,
  colHeaders: true,
  height: 'auto',
  width: 'auto',
  columns: function(column) {
    let columnMeta = {};

    if (column === 0) {
      columnMeta.data = 'id';
    } else if (column === 1) {
      columnMeta.data = 'name.first';
    } else if (column === 2) {
      columnMeta.data = 'name.last';
    } else if (column === 3) {
      columnMeta.data = 'address';
    } else {
      columnMeta = null;
    }

    return columnMeta;
  },
  minSpareRows: 1,
  licenseKey: 'non-commercial-and-evaluation'
});

              具有列映射的对象数组

              在具有嵌套对象的情况下，您可以通过使用columns选项映射列来将它们用作数据源.

                import Handsontable from 'handsontable';
import 'handsontable/dist/handsontable.full.min.css';

const container = document.querySelector('#example5');
const data = [
  { id: 1, name: {first: 'Ted', last: 'Right'}, address: '' },
  { id: 2, address: '' }, // Handsontable will create missing properties on demand
  { id: 3, name: {first: 'Joan', last: 'Well'}, address: '' }
];

const hot = new Handsontable(container, {
  data,
  colHeaders: true,
  height: 'auto',
  width: 'auto',
  columns: [
    { data: 'id' },
    { data: 'name.first' },
    { data: 'name.last' },
    { data: 'address' }
  ],

                具有自定义数据模式的对象数组

                当使用对象数据绑定时，Handsontable 需要知道添加新行时要创建什么数据结构。如果您的数据源至少包含一行，Handsontable 将根据第一行计算出数据结构.

                如果一开始使用的是空数据源，则需要提供数据模式选项，其中包含添加到表格中的新行的数据结构。下面的示例显示了一个带有空数据源的自定义数据模式: 

                  import Handsontable from 'handsontable';
import 'handsontable/dist/handsontable.full.min.css';

const container = document.querySelector('#example6');
const hot = new Handsontable(container, {
  data: [],
  dataSchema: { id: null, name: { first: null, last: null }, address: null },
  startRows: 5,
  startCols: 4,
  colHeaders: ['ID', 'First Name', 'Last Name', 'Address'],
  height: 'auto',
  width: 'auto',
  columns: [
    { data: 'id' },
    { data: 'name.first' },
    { data: 'name.last' },
    { data: 'address' }
  ],
  minSpareRows: 1,
  licenseKey: 'non-commercial-and-evaluation'
});

                  函数数据源和架构

                  如果您的dataSchema是一个不直接暴露其成员的对象的构造函数，您可以为每个列的数据成员指定函数.

                  下面的示例展示了如何使用此类对象:

                    import Handsontable from 'handsontable';
import 'handsontable/dist/handsontable.full.min.css';

const container = document.querySelector('#example7');
const hot = new Handsontable(container, {
  data: [
    model({ id: 1, name: 'Ted Right', address: '' }),
    model({ id: 2, name: 'Frank Honest', address: '' }),
    model({ id: 3, name: 'Joan Well', address: '' }),
    model({ id: 4, name: 'Gail Polite', address: '' }),
    model({ id: 5, name: 'Michael Fair', address: '' })
  ],
  dataSchema: model,
  height: 'auto',
  width: 'auto',
  colHeaders: ['ID', 'Name', 'Address'],
  columns: [
    { data: property('id') },
    { data: property('name') },
    { data: property('address') }
  ],
  minSpareRows: 1,
  licenseKey: 'non-commercial-and-evaluation'
});

function model(opts) {
  let _pub = {
    id: undefined,
    name: undefined,
    address: undefined
  };
  let _priv = {};

  for (const i in opts) {
    if (opts.hasOwnProperty(i)) {
      _priv[i] = opts[i];
    }
  }

  _pub.attr = function (attr, val) {
    if (typeof val === 'undefined') {
      window.console && console.log('GET the', attr, 'value of', _pub);

      return _priv[attr];
    }

    window.console && console.log('SET the', attr, 'value of', _pub);
    _priv[attr] = val;

    return _pub;
  };

  return _pub;
}

function property(attr) {
  return function (row, value) {
    return row.attr(attr, value);
  }
}

                    没有数据

                    默认情况下，如果不提供任何数据，Handsontable 会渲染为一个空的 5x5 表格.

                      import Handsontable from 'handsontable';
import 'handsontable/dist/handsontable.full.min.css';

const container = document.querySelector('#example9');
const hot = new Handsontable(container, {
  licenseKey: 'non-commercial-and-evaluation'
});

                      要更改默认呈现的行数或列数，请使用 startRowsstartRows和startCols选项.

                      数据处理 API 方法

                      了解作为参考的约束力

                      Handsontable 通过引用而不是值与数据源绑定。我们不会复制输入数据集，而是依靠 JavaScript 来处理对象。任何输入到表格中的数据都会改变原始数据源.

                      TIP

                      Handsontable 使用引用初始化表的源数据，但不应依赖它。例如，你不应该使用输入数据集的引用来更改源数据中的值。有些处理数据的机制并不支持以这种方式进行的外部更改.

                      为避免出现这种情况，请在将数据传递给表格之前复制数据。要从 Handsontable 外部更改数据，可以使用我们的 API 方法。例如，在调用setDataAtCell()方法后，正在进行的更改将立即显示在屏幕上.

                        import Handsontable from 'handsontable';
import 'handsontable/dist/handsontable.full.min.css';

const container = document.querySelector('#example10');
const data = [
  ['', 'Tesla', 'Nissan', 'Toyota', 'Honda', 'Mazda', 'Ford'],
  ['2017', 10, 11, 12, 13, 15, 16],
  ['2018', 10, 11, 12, 13, 15, 16],
  ['2019', 10, 11, 12, 13, 15, 16],
  ['2020', 10, 11, 12, 13, 15, 16],
  ['2021', 10, 11, 12, 13, 15, 16]
];
const settings = {
  data: data,
  height: 'auto',
  licenseKey: 'non-commercial-and-evaluation'
};

const hot = new Handsontable(container, settings);

hot.setDataAtCell(0, 1, 'Ford');

                        有多种方法可以将数据插入 Handsontable。让我们来看看最有用的几种:

                        data配置选项

                        您可能需要用一些数据来初始化表格（如果不这样做，表格就会呈现一个空的 5x5 表格）。最简单的方法是在初始配置对象中将数据数组作为data选项传入:

                        const hot = new Handsontable(container, {
  data: newDataset,
  // ... 其他配置选项
});

                        数据加载 API 方法

                        要替换已初始化的 Handsontable 实例中的全部数据，可以使用其中一种数据加载 API 方法:

                        • loadData()
                          用方法参数中提供的数据集替换 Handsontable 中使用的数据.
                          备注: 自12.0.0版起，该方法会导致表重置其配置选项和索引映射器信息，因此表自初始化以来所做的部分工作可能会丢失.
                          hot.loadData(newDataset);
                        • updateData()
                          将Handontable中使用的数据替换为作为方法参数提供的数据集。 与 loadData()不同, updateData() 不会重置配置选项和/或索引映射程序信息，因此可以安全地使用它来仅替换数据，而保持表的其余部分不变.
                          hot.updateData(newDataset);
                        • updateSettings()
                          更新表格配置, updateSettings() 也可以用来替换正在使用的数据。自版本12.0.0以来，它在后台使用updateData()方法来执行数据替换(除了在初始化期间执行的一个自动调用外，它使用loadData()).
                          hot.updateSettings({
  data: newDataset,
  // ... 其他配置选项
});

                        数据修改API方法

                        要只修改传递给Handontable的数据子集，您可能需要检查以下方法:

                        • setDataAtCell()
                          替换单个单元格中的数据或执行一系列单元格数据替换:

                          // 用提供的值替换（0, 2）可视坐标（0 为可视行索引，2 为可视列索引）处的单元格内容。
hot.setDataAtCell(0, 2, 'New Value');

// 用提供的值替换`(0,2)`、`(1,2)`和`(2,2)`处的单元格。
const changes = [
  [0, 2, 'New Value'],
  [1, 2, 'Different Value'],
  [2, 2, 'Third Replaced Value'],
];
hot.setDataAtCell(changes);

                        • setDataAtRowProp()
                          替换单个单元格中的数据或执行一系列单元格数据替换，类似于setDataAtCell(), 但允许通过可视行索引和数据行属性定位单元格. 对Array of objects data type有用.

                          //用提供的值替换（0，'title'）坐标（0是可视行索引，'title'是数据行对象属性）处的单元格内容。
hot.setDataAtRowProp(0, 'title', 'New Value');

//用提供的值替换第一行中具有“id”、“firstName”和“lastName”道具的单元格。
const changes = [
  [0, 'id', '22'],
  [0, 'firstName', 'John'],
  [0, 'lastName', 'Doe'],
];
hot.setDataAtRowProp(changes);

                        • setSourceDataAtCell()
                          由于显示的数据坐标可能与内部存储的方式不同，有时您可能需要更直接地针对单元格 - 这就是setSourceDataAtCell() 派上用场的时候. row 和 columns/prop 参数表示物理索引.

                          // 用提供的值替换 (0, 2) 坐标（0 为物理行索引，2 为物理列索引）上的单元格内容。
hot.setSourceDataAtCell(0, 2, 'New Value');

//用提供的值替换（0，'title'）坐标（0是物理行索引，'title'是数据行属性）处的单元格内容。
hot.setSourceDataAtCell(0, 'title', 'New Value');

// 用提供的值替换第一物理行中包含 "id"、"firstName "和 "lastName "道具的单元格。
const changes = [
  [0, 'id', '22'],
  [0, 'firstName', 'John'],
  [0, 'lastName', 'Doe'],
];
hot.setSourceDataAtCell(changes);

                        • populateFromArray()
                          替换数据集的一个数据块，提供起始坐标（也可提供终止坐标）和包含新值的二维数据数组.

                          TIP

                          populateFromArray() 方法不能更改只读单元格.

                          const newValues = [
  ['A', 'B', 'C'],
  ['D', 'E', 'F']
];

// 用 `newValues` 数组中的值替换 (1, 1) 到 (2, 3) 可视化单元格坐标值。
hot.populateFromArray(1, 1, newValues);

//  用 `newValues` 数组中的值替换 (1, 1) 到 (2, 2) 可视化单元格坐标中的值，删除超出定义范围的值。
hot.populateFromArray(1, 1, newValues, 2, 2);

                        使用数据副本

                        使用Handontable的数据副本时，最佳做法是在将数据源加载到Handontable之前克隆数据源。这可以通过JSON.parse(JSON.stringify(data))或其他深度克隆函数来完成. 

                          import Handsontable from 'handsontable';
import 'handsontable/dist/handsontable.full.min.css';

const container = document.querySelector('#example11');
const data = [
  ['', 'Tesla', 'Nissan', 'Toyota', 'Honda', 'Mazda', 'Ford'],
  ['2017', 10, 11, 12, 13, 15, 16],
  ['2018', 10, 11, 12, 13, 15, 16],
  ['2019', 10, 11, 12, 13, 15, 16],
  ['2020', 10, 11, 12, 13, 15, 16],
  ['2021', 10, 11, 12, 13, 15, 16]
];

const hot = new Handsontable(container, {
  data: JSON.parse(JSON.stringify(data)),
  height: 'auto',
  licenseKey: 'non-commercial-and-evaluation'
});
