      JavaScript Data Grid 选择

      选择单个单元格、相邻单元格范围或多个非相邻单元格范围。

      概述

      选择功能可让您在 Handsontable 中选择单个单元格或单元格范围。选定后，您可以从单元格中获取数据、编辑单元格内容或更改单元格样式。

      基本配置

      利用该功能，您可以在表格中选择单个单元格或单元格范围。轻松检索所选单元格的坐标，清除或更改单元格内容。

      使用 Mac 的Cmd或 Windows 的Ctrl键选择不相邻的单元格区域。

      选择范围

      您可以在不同模式下使用该插件。您可以选择选择单个单元格、相邻单元格区域或多个非相邻单元格区域。

      selectionMode的可能值：

      • single - 您可以选择一个单元格。
      • range - 您可以在一个选定范围内选择多个单元格。
      • multiple - 您可以选择多个不相邻的单元格区域。

        import Handsontable from 'handsontable';
import 'handsontable/dist/handsontable.full.min.css';

const selectOption = document.querySelector('#selectOption');
const container = document.querySelector('#example1');
const hot = new Handsontable(container, {
  data: [
    ['A1', 'B1', 'C1', 'D1', 'E1', 'F1', 'G1', 'H1', 'I1'],
    ['A2', 'B2', 'C2', 'D2', 'E2', 'F2', 'G2', 'H2', 'I2'],
    ['A3', 'B3', 'C3', 'D3', 'E3', 'F3', 'G3', 'H3', 'I3'],
    ['A4', 'B4', 'C4', 'D4', 'E4', 'F4', 'G4', 'H4', 'I4'],
    ['A5', 'B5', 'C5', 'D5', 'E5', 'F5', 'G5', 'H5', 'I5'],
    ['A6', 'B6', 'C6', 'D6', 'E6', 'F6', 'G6', 'H6', 'I6'],
    ['A7', 'B7', 'C7', 'D7', 'E7', 'F7', 'G7', 'H7', 'I7'],
    ['A8', 'B8', 'C8', 'D8', 'E8', 'F8', 'G8', 'H8', 'I8'],
    ['A9', 'B9', 'C9', 'D9', 'E9', 'F9', 'G9', 'H9', 'I9'],
  ],
  width: 'auto',
  height: 'auto',
  colWidths: 100,
  rowHeights: 23,
  rowHeaders: true,
  colHeaders: true,
  selectionMode: 'multiple', // 'single', 'range' or 'multiple',
  licenseKey: 'non-commercial-and-evaluation'
});

selectOption.addEventListener('change', event => {
  const value = event.target.value;
  const first = value.split(' ')[0].toLowerCase();

  hot.updateSettings({
    selectionMode: first
  });
});

        从选定范围获取数据

        要以数组的形式检索所选单元格，可以使用getSelected()或getSelectedRange()方法。

          import Handsontable from 'handsontable';
import 'handsontable/dist/handsontable.full.min.css';

const output = document.querySelector('#output');
const getButton = document.querySelector('#getButton');
const container = document.querySelector('#example2');
const hot = new Handsontable(container, {
  data: [
    ['A1', 'B1', 'C1', 'D1', 'E1', 'F1', 'G1', 'H1', 'I1'],
    ['A2', 'B2', 'C2', 'D2', 'E2', 'F2', 'G2', 'H2', 'I2'],
    ['A3', 'B3', 'C3', 'D3', 'E3', 'F3', 'G3', 'H3', 'I3'],
    ['A4', 'B4', 'C4', 'D4', 'E4', 'F4', 'G4', 'H4', 'I4'],
    ['A5', 'B5', 'C5', 'D5', 'E5', 'F5', 'G5', 'H5', 'I5'],
    ['A6', 'B6', 'C6', 'D6', 'E6', 'F6', 'G6', 'H6', 'I6'],
    ['A7', 'B7', 'C7', 'D7', 'E7', 'F7', 'G7', 'H7', 'I7'],
    ['A8', 'B8', 'C8', 'D8', 'E8', 'F8', 'G8', 'H8', 'I8'],
    ['A9', 'B9', 'C9', 'D9', 'E9', 'F9', 'G9', 'H9', 'I9'],
  ],
  width: 'auto',
  height: 'auto',
  colWidths: 100,
  rowHeights: 23,
  rowHeaders: true,
  colHeaders: true,
  outsideClickDeselects: false,
  selectionMode: 'multiple', // 'single', 'range' or 'multiple',
  licenseKey: 'non-commercial-and-evaluation'
});

getButton.addEventListener('click', event => {
  const selected = hot.getSelected() || [];
  const data = [];

  for (let i = 0; i < selected.length; i += 1) {
    const item = selected[i];

    data.push(hot.getData(...item));
  }

  output.innerText = JSON.stringify(data);
});

          修改选定的单元格

          您可能想删除、格式化或以其他方式更改所选单元格。例如，您可以使用下面的演示来更改数值或为选定单元格添加 CSS 类。

            import Handsontable from 'handsontable';
import 'handsontable/dist/handsontable.full.min.css';

const button = document.querySelector('#set-data-action');
const container = document.querySelector('#example3');
const hot = new Handsontable(container, {
  data: [
    ['A1', 'B1', 'C1', 'D1', 'E1', 'F1', 'G1', 'H1', 'I1'],
    ['A2', 'B2', 'C2', 'D2', 'E2', 'F2', 'G2', 'H2', 'I2'],
    ['A3', 'B3', 'C3', 'D3', 'E3', 'F3', 'G3', 'H3', 'I3'],
    ['A4', 'B4', 'C4', 'D4', 'E4', 'F4', 'G4', 'H4', 'I4'],
    ['A5', 'B5', 'C5', 'D5', 'E5', 'F5', 'G5', 'H5', 'I5'],
    ['A6', 'B6', 'C6', 'D6', 'E6', 'F6', 'G6', 'H6', 'I6'],
    ['A7', 'B7', 'C7', 'D7', 'E7', 'F7', 'G7', 'H7', 'I7'],
    ['A8', 'B8', 'C8', 'D8', 'E8', 'F8', 'G8', 'H8', 'I8'],
    ['A9', 'B9', 'C9', 'D9', 'E9', 'F9', 'G9', 'H9', 'I9'],
  ],
  width: 'auto',
  height: 272,
  colWidths: 100,
  rowHeights: 23,
  rowHeaders: true,
  colHeaders: true,
  outsideClickDeselects: false,
  selectionMode: 'multiple', // 'single', 'range' or 'multiple',
  licenseKey: 'non-commercial-and-evaluation'
});

button.addEventListener('click', event => {
  const selected = hot.getSelected() || [];
  const target = event.target.id;

  hot.suspendRender();

  for (let index = 0; index < selected.length; index += 1) {
    const [row1, column1, row2, column2] = selected[index];
    const startRow = Math.max(Math.min(row1, row2), 0);
    const endRow = Math.max(row1, row2);
    const startCol = Math.max(Math.min(column1, column2), 0);
    const endCol = Math.max(column1, column2);

    for (let rowIndex = startRow; rowIndex <= endRow; rowIndex += 1) {
      for (let columnIndex = startCol; columnIndex <= endCol; columnIndex += 1) {
        hot.setDataAtCell(rowIndex, columnIndex, 'data changed');
        hot.setCellMeta(rowIndex, columnIndex, 'className', 'c-red');
      }
    }
  }

  hot.render();
  hot.resumeRender();
});

            选择区域的样式

            您可以使用 CSS 样式轻松更改背景颜色。淡蓝色的主要背景颜色在 .area 类中定义。

            对于非相邻的选择，多个等级会使每个等级的颜色更深一些。这些类别被称为 area-1, area-2等。

            遗憾的是，没有简单的方法来更改选区的边框颜色。

            跳过表格的边缘

            使用键盘导航跨越表格边缘时，可以设置单元格选择跳转到对边。

            跳过垂直边缘

            启用左右边缘跳转：

            • 将autoWrapRow配置选项设置为true。

            跳过垂直边缘

            • 当单元格选择位于一行的第一个单元格时，按左箭头键。
            • 当单元格选中一行的最后一个单元格时，按向右箭头键，或按Tab键。

            跳过水平边缘

            启用上下边缘跳转：

            • 将autoWrapCol配置选项设置为 true。

            跳过水平边缘

            • 当单元格选择位于列的第一个单元格时，按向上箭头键。
            • 当单元格选择位于列的最后一个单元格时，按向下箭头键或按Enter键。
