      概述

      通过排序，您可以根据特定列中的值轻松地重新排列数据行。这对于分析和整理大型数据集特别有用，可以帮助你识别模式和趋势。

      您可以通过不同的方式对数据进行排序：

      • 按字母、数字或自定义排序逻辑进行排序
      • 以升序、降序或自定义顺序排列
      • 按单列或多列计算
      • 使用 Handsontable 的用户界面或应用程序接口

      Handsontable 仅对数据进行可视化排序，因此源数据仍保持原有顺序。要在数据源中保存排序更改，请参阅本指南：保存数据。

        启用排序

        要启用所有列的排序，请将 columnSorting 设置为 true。

        const configurationOptions = {
  // 启用所有列的排序
  columnSorting: true,
};

          配置排序

          您可以配置排序用户界面，设置初始排序顺序，并实现自己的比较器。

          默认情况下:

          • 所有列均已启用排序。
          • 最终用户可以通过单击列名对数据进行排序。
          • 排序顺序指示器可见。
          • 在 Handsontable 初始化时，不会对行进行排序。

          您可以配置以下选项：

          const configurationOptions = {
  columnSorting: {
    // 让最终用户通过点击列名对数据进行排序（默认设置）
    headerAction: true,
    // 不对空单元格进行排序 - 将包含空单元格的行移到底部（默认设置）
    sortEmptyCells: false,
    // 启用列名旁边的排序顺序图标（默认设置）
    indicator: true,

    // 初始化时，按第一列降序排列数据
    initialConfig: {
      column: 1,
      sortOrder: 'desc',
    },

    // 实现自己的比较器
    compareFunctionFactory(sortOrder, columnMeta) {
      return function (value, nextValue) {
        // 在这里，添加一个比较函数
        // 返回 `-1`，或 `0`，或 `1
      };
    },
  },
};

          • time

              const configurationOptions = {
  // 启用所有列的多列排序
  multiColumnSorting: true,
};

              要选择哪些列可以同时排序，请将不想排序的列的headerAction设置为false。 

              const configurationOptions = {
  // 启用多列排序，适用于所有列
  multiColumnSorting: true,

      multiColumnSorting: {

      },
    },

              columnSorting和multiColumnSorting选项相互覆盖。如果同时使用这两个选项，则后面定义的选项优先。

                      添加自定义比较器

                      比较器是一种根据指定条件确定排序顺序的函数。

                      添加自定义比较器可以让您超越Handsontable的内置排序功能。您可以：

                      • 应用自定义排序顺序。例如，您可以按长度或特定字符的出现次数排序，而不是按字母或数字排序。
                      • 处理例外情况。例如，在员工列表中，可以将具有特定职称的员工排除在排序之外。
                      • 根据您自己的标准实现自定义排序逻辑。

                      要添加自定义比较器，请使用compareFunctionFactory选项。

                      const configurationOptions = {
  columnSorting: {
    compareFunctionFactory: function (sortOrder, columnMeta) {
      // 实现自己的比较器
      return function (value, nextValue) {
        if (value < nextValue) {
          return -1;
        }
        if (value > nextValue) {
          return 1;
        }

        return 0;
      };
    },
  },
};

                      使用排序钩子

                      您可以使用以下 Handsontable 钩子，在排序之前或之后运行代码：

                      • beforeColumnSort()
                      • afterColumnSort()

                      例如，可以使用beforeColumnSort()进行服务器端排序，或使用afterColumnSort()从排序中排除行。

                      const configurationOptions = {

  },

                        以编程方式控制排序

                        您可以使用 Handsontable 的钩子和 API 方法在表格运行时控制排序。

                        这样您就可以：

                        • 根据指定条件启用或禁用排序。例如，您可以禁用超大数据集的排序。
                        • 根据应用程序中另一个组件的状态触发排序。例如，您可以让终端用户通过点击表格外的按钮对数据进行排序。

                        以编程方式启用或禁用排序

                        要以编程方式启用或禁用排序，请使用updateSettings()方法。

                        // 启用所有列的排序
handsontableInstance.updateSettings({
  columnSorting: true,
});

// 禁用所有列的排序
handsontableInstance.updateSettings({
  columnSorting: false,
});

                        您还可以启用或禁用特定列的排序。

                        handsontableInstance.updateSettings({
  columns: [
    {
      // 启用第一列的排序
      columnSorting: {
        headerAction: true,
      },
    },
    {
      // 禁用第二列的排序
      columnSorting: {
        headerAction: false,
      },
    },
  ],
});

                        以编程方式排序数据

                        要以编程方式对数据进行排序，请使用columnSorting.sort()方法。请记住先启用排序。

                        注意，调用columnSorting.sort()会覆盖之前的排序顺序。 

                        const configurationOptions = {
  // 启用所有列的排序
  columnSorting: true,
};

const columnSorting = handsontableInstance.getPlugin('columnSorting');

columnSorting.sort(
  // 按第一列升序对数据进行排序
  {
    column: 0,
    sortOrder: 'asc', // 如果是降序，使用`'desc'`
  }
);

// 回到原来的顺序
columnSorting.clearSort();

                            导入排序模块

                            通过仅导入和注册所需的模块，可以减小包的大小。

                            要使用排序功能，您只需要以下模块：

                            • 基础模块
                            • 列排序模块或 多列排序模块 
                            // 导入基础模块
import Handsontable from 'handsontable/base';

// 导入 Handsontable 的 CSS
import 'handsontable/dist/handsontable.full.min.css';

// 导入 ColumnSorting 插件（或 MultiColumnSorting 插件）
import { registerPlugin, ColumnSorting } from 'handsontable/plugins';

// 注册列排序（或多列排序）插件
registerPlugin(ColumnSorting);
